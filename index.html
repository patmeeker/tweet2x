Tweet2X for Google Chrome is available for download at the Chrome web store.

Browser Support

Tweet2X is currently available for Chrome only, but the first thing on my to do list, is to package it for FireFox and then for Safari. I will be tweeting regarding any changes in the status of the project via my personal account (patmeeker) on Twitter, so feel free to stay tuned for that info.

Language Support

Tweet2X currently supports the following characters only ABDEFGHIJKLMNOPQRSTUVWXYZ abcdefghijklmnopqrstuvwxyz 0123456789,.?!'"@#$%()-_ However, I plan to add support for other languages in the future. If you'd like to let me know via Twitter, which other language would be most useful for you, I'll get started working on the most popular demand, and ofcourse tweet any updates for you.
